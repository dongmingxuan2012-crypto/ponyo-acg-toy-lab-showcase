# 波妞 PONYO（原名董冰）｜IP 品牌 × AIGC 视觉叙事创作者

波妞 PONYO，原名董冰，1983 年 5 月 26 日生于黑龙江哈尔滨。1999 年就读于中央音乐学院，毕业后在煤矿文工团任演员，后入职华音音乐，担任艺人及音乐人。早期的音乐、舞台和艺人经历，让她长期训练对角色气质、情绪表达、镜头感和大众传播的判断。

之后进入 IP 品牌与授权行业，曾任职《小猪佩奇》中国区市场团队，深度参与该 IP 在中国市场的品牌营销推广与授权业务，积累了从 UGC 内容激活、PGC 内容生产到 IP 授权商业化落地的全链路实战经验，理解顶级亲子 IP 如何在本土市场建立规模化影响力。

目前，PONYO 专注 AI 视觉叙事、游戏 IP 潮玩化、动漫角色出镜、漫画 IP 孵化和高阶 Prompt 工作流，能够把一个角色从概念设定推进到样片、广告包装、创作者模板和商业展示页，让创意从“好看的画面”进入“可展示、可发布、可复用、可招商”的内容资产阶段。

Web3 与 RWA 方向作为补充背景，PONYO 曾参与卡牌游戏项目社群推广与用户运营，并沉淀过较大规模的自有社群经验。这部分能力主要服务于对年轻用户参与机制、社区扩散和内容共创路径的理解。

## 合作价值

- 游戏 IP 潮玩化与现实直播流样片
- AIGC 创作者挑战模板包
- IP 联名、快闪、授权招商视觉预演
- 社媒传播短片与商业展示页包装
- 从角色设定、分镜故事板、视频提示词到成片包装的 AI 内容生产链路
